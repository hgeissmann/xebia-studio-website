SIMPLICITÉ

 


Il est souvent inutile de surcharger votre applicatif de frameworks plus ou moins simples qui apportent autant de problèmes de contournement qu'il ne cadre véritablement votre projet. Un projet réussi c'est avant tout des fonctions utiles à ses utilisateurs. La technologies se doit d'être au service du projet et non pas l'inverse.

Votre projet n'est pas un laboratoire de R&D, ni une période d'apprentissage pour nos développeurs. Nous utilisons des technologies que nous avons déjà éprouvées, avec votre accord.

WEB & MULTI-PLATEFORME

 


Ces dernières années, nous nous sommes habitués à vivre dans un monde 100% connecté. Nos applications sont accessibles depuis tous les terminaux, WEB, mobiles, tablettes.



CLOUD

 


En 2012, le cloud est une réalité. Il apporte son lot de souplesse et de rapidité avec une série de contraintes que nous maîtrisons. Tous les projets ne sont pas éligibles à être dans le cloud.

Nous utilisons et vivons dans le cloud pour plusieurs de nos projets. Nous dispensons des ateliers pour partager notre savoir faire.

Nous pouvons utiliser le cloud pour l'hébergement de votre solution (Amazon EC2, Heroku, Ovh Cloud) ou plus simplement pour l'infrastructure projet grâce à notre partenariat avec CloudBees.

NoSQL : NOT ONLY SQL

 


Les technologies dites NoSQL sont en rupture avec les traditionnelles bases de données que nous utilisons depuis des dizaines d'années. Leur avantage se situe aussi dans leur souplesse d'usage. Nous avons plusieurs projets en productions utilisant se types de base de stockage de données (MongoDB, Redis, HBase/Hadoop).

Avec votre accord nous sommes a même d'apporter conseil et recul sur l'emploi de ses technologies.
